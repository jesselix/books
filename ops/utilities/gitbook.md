# GitBook

-> 安装好node.js

-> 安装
npm install -g gitbook-cli

-> 检查版本
gitbook -V

-> 找一个目录创建一本书的文件夹，比如：z-demo-book
mkdir z-demo
cd z-demo

-> 浏览器预览
gitbook serve
在浏览器中输入“http://localhost:4000”即可预览电子书内容，预览完毕后按Ctrl + C结束。

-> 或者直接构建
gitbook build
